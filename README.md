# ✍️ AirScript AI

> **Write in the Air. Powered by AI.**

AirScript AI is an AI-powered air writing application that enables users to write letters and words by moving their fingers in the air. Using Computer Vision, Hand Tracking, and Deep Learning, the system captures hand gestures through a webcam, recognizes handwritten characters, predicts words intelligently, and converts them into editable digital text in real time.

This project demonstrates the integration of Artificial Intelligence, Computer Vision, and Human-Computer Interaction (HCI) to create a touchless writing experience.

---

## 🚀 Features

- ✋ Real-time hand tracking using a webcam
- ✍️ Air writing with fingertip gestures
- 🔤 AI-based handwritten character recognition
- 🤖 Intelligent word prediction
- 📝 Sentence formation
- 📚 Writing history
- 💾 Save generated text
- 🧹 Clear writing canvas
- 🎨 User-friendly graphical interface
- ⚡ Fast real-time performance


---

## 🧠 How It Works

1. Captures hand movements using the webcam.
2. Detects hand landmarks using MediaPipe.
3. Tracks the fingertip path for air writing.
4. Converts the drawn trajectory into an image.
5. Uses a trained AI model to recognize handwritten characters.
6. Predicts the intended word using AI.
7. Displays editable text in real time.
8. Allows users to save or clear the generated content.

## 📦 Installation


